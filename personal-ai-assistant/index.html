🔄 Request Lifecycle
Calendar Query
Send Email
Cross-Service

"What's on my calendar tomorrow?"

1

User sends message

Message arrives via FastAPI / ADK Web UI

2

Coordinator classifies intent

Gemini: "This is a calendar query" → delegates to Calendar Agent

3

Calendar Agent selects tool

Calls calendar_list_events(user_id, time_min="tomorrow")

4

MCP Server processes

Loads user's OAuth token → calls Google Calendar API → returns events

5

Natural language response

"You have 3 meetings tomorrow: Team Standup at 10 AM, Design Review at 2 PM, 1:1 with Manager at 4 PM"

"Send an email to John about the project update"

1

Coordinator → Gmail Agent

Intent: email composition

2

Agent drafts email content

Gemini generates professional email body based on context

3

MCP tool: gmail_send_email

to: john@company.com, subject: "Project Update", body: ...

4

Confirmation

"✅ Email sent to John about the project update"

"Read my latest email and schedule a meeting based on it"

1

Coordinator delegates first to Gmail Agent

Reads the latest email → extracts meeting request details

2

Coordinator then delegates to Calendar Agent

Uses extracted info to create a new calendar event

3

Combined response

"I read the email from Sarah about the quarterly review. I've scheduled it for Friday at 3 PM."

🚀 Quick Start
1

Ensure MCP server is running

cd ../mcp-servers/google_workspace source .venv/bin/activate MCP_TRANSPORT=streamable-http python server.py
2

Start the AI Assistant

source .venv/bin/activate adk web # ADK dev UI # or python -m app.main # FastAPI server
3

Chat with your assistant

Try: "What's on my schedule today?", "Search emails from Google", "List my Drive files"

📦 Tech Stack

🧠 Google ADK

Agent Development Kit for multi-agent orchestration with tool use, delegation, and Gemini model integration.

⚡ Gemini 2.5 Flash

Google's latest LLM for fast, accurate reasoning. Powers intent classification and response generation.

🔌 MCP SDK

Model Context Protocol for standardized tool access. Connects agents to external services via Streamable HTTP.

🌐 FastAPI

Modern Python web framework for serving the agent system via HTTP endpoints and SSE streaming.

🐍 Python 3.12

Isolated virtual environment with UV package manager for fast, reproducible dependency management.

📋 Pydantic

Data validation and settings management for type-safe agent configuration and request handling.
